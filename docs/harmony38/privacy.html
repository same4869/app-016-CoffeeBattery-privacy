隐私保护政策

最后更新日期：2025年5月12日 | 生效日期：2025年5月12日

网购后悔研究所（以下简称"本应用"）由独立开发者开发和运营。我们深知个人信息对您的重要性，并会尽全力保护您的隐私安全。请您在使用本应用前，仔细阅读并了解本隐私政策。

一、数据收集范围

本应用仅记录您主动输入的以下信息：

  • 商品名称（代号）
  • 购买金额
  • 商品品类
  • 消费诱因
  • 冲动指数评分（1-10）
  • 后悔度评分（1-5）
  • 使用频率评估
  • 二手残值预估

以上所有数据均为用户自愿填写。我们不会自动采集任何设备信息、个人身份信息、位置信息或其他隐私数据。

二、数据存储方式

所有数据均存储在您的设备本地（应用沙箱目录），不会上传至任何服务器或云端。

本应用无需网络权限，不包含任何网络请求功能。即使在联网环境下，应用也不会发送任何数据。

三、数据安全

您的数据受到 HarmonyOS 应用沙箱机制保护，其他应用无法访问本应用的数据。数据库采用 S1 安全级别加密存储。

四、第三方服务

本应用不集成任何第三方 SDK，包括但不限于：

  • 广告框架
  • 统计分析工具
  • 社交分享组件
  • 推送通知服务
  • 崩溃收集工具

应用为纯本地运行，零依赖外部服务。

五、权限申请

本应用不请求任何系统权限，包括但不限于：通讯录、相册、定位、摄像头、麦克风、存储（外部）、网络等。

我们奉行"零权限"运营理念，最大化保障用户隐私。

六、用户权利

您可以随时行使以下权利：

  • 数据导出：通过"设置 → 导出本地数据快照"将所有数据导出为 JSON 格式文件
  • 数据删除：通过"设置 → 清除所有实验记录"永久删除所有数据
  • 单条删除：长按任意商品卡片可删除单条记录
  • 完全清除：卸载应用即可完全清除设备上的所有本地数据

七、未成年人保护

本应用不面向14周岁以下的未成年人。如果您是未成年人的监护人，发现您的孩子在使用本应用，请联系我们，我们将协助处理。

八、隐私政策更新

我们可能会不时更新本隐私政策。更新后的政策将通过应用版本更新的方式通知您。重大变更时，我们会在应用内以弹窗等显著方式提示您重新阅读。

九、适用法律

本隐私政策适用中华人民共和国相关法律法规，包括《中华人民共和国个人信息保护法》《中华人民共和国数据安全法》及《中华人民共和国网络安全法》。
