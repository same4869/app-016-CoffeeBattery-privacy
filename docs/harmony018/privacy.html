FIRE自由倒计时 隐私政策

最后更新日期：2026年5月4日

FIRE自由倒计时（以下简称"本应用"）由开发者 xun 开发并运营。我们深知个人财务数据的敏感性，因此本应用采用纯本地运算架构，不收集、不上传、不存储任何用户数据至远程服务器。

一、数据存储方式

本应用的所有数据（包括但不限于个人财务参数、资产记录、使用偏好）均存储在您的设备本地，使用 HarmonyOS 系统提供的 Preferences API 进行持久化存储。数据不会离开您的设备。

二、信息收集

本应用不收集任何个人信息，具体包括：

  • 不收集姓名、手机号、身份证号等身份信息
  • 不收集地理位置信息
  • 不收集设备标识符或广告 ID
  • 不收集使用行为数据或浏览记录
  • 不存在用户账号体系和注册登录流程
  • 不发起任何网络请求

三、数据安全

您的数据受到 HarmonyOS 系统级应用沙箱机制保护，其他应用无法访问本应用的数据目录。

卸载本应用后，所有本地数据将被系统永久删除，不可恢复。

您也可以在应用内"我的 → 清除所有数据"中手动擦除全部记录。

四、第三方服务

本应用不集成任何第三方服务，包括但不限于：

  • 无第三方统计分析 SDK（如友盟、Firebase 等）
  • 无广告网络 SDK
  • 无社交分享 SDK
  • 无推送服务 SDK
  • 无崩溃收集 SDK

五、权限使用说明

本应用仅申请以下权限：

  • ohos.permission.VIBRATE（振动）：用于微储蓄按钮的触觉反馈，提升交互体验。该权限为系统级权限，无需用户手动授权。

本应用不申请以下敏感权限：

  • 网络访问权限
  • 位置信息权限
  • 相机/麦克风权限
  • 通讯录/通话记录权限
  • 文件读写权限

六、儿童隐私保护

本应用不面向14周岁以下的未成年人设计或推广。本应用不主动收集任何年龄段用户的个人信息。

七、数据共享与转让

由于本应用不收集任何用户数据，因此不存在数据共享、转让或公开披露的情形。

八、用户权利

您对本应用中的数据享有完全控制权：

  • 查看：所有数据均在应用界面中直接展示
  • 修改：您可以随时在"个人档案设置"中修改任何参数
  • 删除：您可以随时通过"清除所有数据"功能一键删除全部记录
  • 导出：卸载前请自行记录重要数据（当前版本暂不支持导出功能）

九、隐私政策更新

如本隐私政策发生变更，我们将通过应用更新的方式通知您。重大变更将在应用内以弹窗形式告知。

我们建议您在每次应用更新后查阅最新版本的隐私政策。
