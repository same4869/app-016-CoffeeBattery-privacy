二、我们如何收集信息

本应用不收集任何个人信息。

本应用为纯本地工具类产品，所有数据（包括您创建的思维节点、连线关系、标签、系统日志等）均存储在您设备本地的应用沙箱中，不会通过任何方式上传至服务器或第三方。

本应用不包含：

  • 用户账号/登录系统
  • 网络数据传输功能
  • 第三方统计/分析 SDK
  • 广告服务
  • 社交/分享功能

三、权限使用说明

本应用申请的系统权限及用途如下：

  • 振动（Vibrate）：用于操作交互的触觉反馈（如创建节点、建立连线时的短振动），提升使用体验。此权限不涉及任何个人信息。

本应用不申请以下敏感权限：网络访问、位置信息、相机、麦克风、通讯录、存储（沙箱外）、电话状态等。

四、数据存储与安全

  • 所有数据以结构化格式存储在应用本地沙箱的关系型数据库中
  • 数据仅在您的设备上存在，开发者无法访问
  • 您可随时通过应用内「系统演化 → 导出镜像」功能，将全部数据导出为 JSON 格式备份至剪贴板
  • 卸载应用将永久删除所有本地数据，且不可恢复

五、数据共享与披露

本应用不与任何第三方共享您的数据。因为数据从未离开您的设备，不存在数据共享的技术可能性。

六、未成年人保护

本应用不针对未成年人收集任何信息。由于本应用不具备网络功能且不收集个人信息，不存在未成年人信息保护风险。

七、隐私政策的变更

我们可能适时修订本隐私政策。政策变更时，我们会在应用内或本页面发布更新版本并标注生效日期。重大变更会在您下次打开应用时以弹窗形式通知。
