隐私政策

VibeCast 状态气象局 · 最后更新：2026年5月

VibeCast（状态气象局）是一款纯本地运行的个人状态观测工具。我们深知隐私对您的重要性，因此在产品设计之初就将「零数据外传」作为核心原则。

数据存储

您的所有观测记录、季节归档及偏好设置仅存储在您设备的本地数据库中。我们不运营任何服务器，不收集、不上传、不备份您的任何个人数据。卸载应用即彻底、永久地删除所有记录。

网络通信

本应用不包含任何网络请求代码。具体而言：

  • 无后端 API 调用
  • 无云端数据同步
  • 无远程分析或崩溃上报
  • 无广告 SDK
  • 无第三方统计服务

应用在完全断网的环境下可正常使用全部功能。

权限使用

本应用仅申请以下权限：

  • 震动（VIBRATE）：用于滑块交互和封存操作时的触觉反馈，可在应用设置中随时关闭。

本应用不申请且不使用以下敏感权限：

  • 定位 / GPS
  • 相机 / 麦克风
  • 通讯录 / 通话记录
  • 文件存储（公共目录）
  • 后台运行

用户追踪

本应用不含任何形式的用户追踪机制：

  • 不收集设备标识符（OAID、IMEI 等）
  • 不进行行为分析或用户画像
  • 不使用广告标识符
  • 不设账号体系，无需注册或登录

第三方服务

本应用不集成任何第三方 SDK、分析工具、广告平台或社交分享组件。所有功能均使用 HarmonyOS 系统原生能力实现。

数据安全

本地数据库使用 HarmonyOS 安全等级 S1 存储，仅本应用进程可访问，不支持跨应用数据共享。您可通过应用内「设置 → 导出数据」功能自主备份，或通过「清除所有数据」功能一键删除。

儿童隐私

本应用不针对 14 岁以下儿童设计，不主动收集任何年龄相关信息。由于应用完全离线且不收集任何数据，不存在儿童个人信息保护风险。

政策变更

如本隐私政策发生变更，我们将通过应用更新的方式通知用户。鉴于本应用的纯本地特性，隐私政策预计不会发生实质性变更。
