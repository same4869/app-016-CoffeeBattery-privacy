  • 应用名称：卷王模拟器

  • 应用性质：娱乐化氛围工具箱（纯本地单机应用）

二、我们不收集任何个人信息

本应用承诺：

  • 不收集您的姓名、手机号、身份证号等个人身份信息
  • 不收集您的位置信息、通讯录、通话记录
  • 不收集您的设备标识符（IMEI、OAID 等）
  • 不收集您的使用行为数据或浏览记录
  • 不收集任何生物特征信息

三、权限说明

本应用不申请任何系统权限，包括但不限于：

  • 不申请网络访问权限（应用完全离线运行）
  • 不申请存储读写权限
  • 不申请相机、麦克风权限
  • 不申请位置定位权限
  • 不申请通知推送权限

四、数据存储

本应用生成的所有数据（包括海报图片、画廊记录、设置偏好）仅存储于您设备本地的应用沙盒目录中：

  • 数据不会上传至任何服务器
  • 数据不会与第三方共享
  • 卸载应用后所有数据将被自动清除

五、第三方 SDK

本应用不集成任何第三方 SDK，包括但不限于：

  • 不集成广告 SDK
  • 不集成统计分析 SDK
  • 不集成社交分享 SDK
  • 不集成推送服务 SDK

六、未成年人保护

本应用内容适合所有年龄段用户。由于本应用不收集任何信息，不存在未成年人个人信息保护的相关风险。

特别提醒：本应用是一款娱乐工具，生成的所有模拟数据均为虚构。请勿将本应用生成的内容用于学业作弊、工作欺骗或其他不当用途。
