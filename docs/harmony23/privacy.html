隐私政策

最后更新日期：2026年5月6日

核心承诺：秒薪不收集、不上传、不存储任何用户数据至远程服务器。您的所有信息仅存在于您的设备中。

一、概述

「秒薪」是一款基于本地优先（Local-first）架构的个人财务认知与情绪管理工具。我们高度重视您的隐私保护，并承诺以最小化原则处理您的个人信息。本政策适用于秒薪应用的所有功能和服务。

二、信息收集

本应用不收集任何个人信息，包括但不限于：

• 设备标识符（IMEI、MAC地址等）

• 地理位置信息

• 通讯录、短信、通话记录

• 相册、相机数据

• 网络浏览记录

• 生物识别信息

您在应用内输入的薪资、资产、消费等财务数据仅用于本地计算和展示，绝不外传。

三、数据存储

所有数据通过 HarmonyOS Preferences API 存储在应用私有沙箱目录中：

• 存储位置：设备本地应用沙箱，其他应用无法访问

• 加密方式：由操作系统级别的沙箱机制保护

• 数据寿命：卸载应用将永久且不可恢复地删除所有数据

四、网络通信

本应用不进行任何网络通信。应用运行完全不依赖网络连接，在飞行模式下亦可正常使用全部功能。本应用不申请网络访问权限。

五、第三方服务

本应用：

• 不集成任何第三方 SDK

• 不包含任何形式的广告

• 不使用任何数据分析或统计服务

• 不接入任何社交平台或分享服务

六、权限说明

本应用不申请任何系统权限。不访问您的相机、麦克风、位置、通讯录、日历、存储（公共目录）或任何其他系统资源。

七、未成年人保护

本应用不针对未成年人设计，不主动收集任何未成年人的个人信息。如果您是14周岁以下的未成年人，请在监护人的陪同和指导下使用本应用。

八、用户权利

由于所有数据均存储在您的设备本地，您拥有对数据的完全控制权：

• 查看：随时在应用内查看所有已录入数据

• 修改：随时在设置中修改任何配置

• 删除：通过设置中的「清除所有数据」功能一键删除，或卸载应用彻底清除

• 导出：当前版本暂不支持数据导出，后续版本可能提供

九、政策变更

如本隐私政策发生重大变更，我们将通过应用内更新说明告知您。变更后继续使用本应用即表示您同意更新后的隐私政策。建议您定期查阅本政策。
