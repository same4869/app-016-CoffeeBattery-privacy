效期管家 隐私政策

更新日期：2026年5月6日  |  生效日期：2026年5月6日

效期管家是一款纯本地应用，不联网、不采集任何个人信息、不包含第三方 SDK。您的所有数据仅存储在您的设备中。

一、我们如何收集和使用信息

效期管家不会收集、上传或共享任何用户个人信息。应用运行过程中产生的所有数据（包括您录入的资产信息、凭证照片、提醒设置等）均仅保存在您的设备本地存储中。

我们不使用任何数据统计、行为分析或广告追踪工具。

二、我们如何使用设备权限

为提供完整的产品功能，应用可能会申请以下设备权限：

  • 相册/存储访问权限：仅在您主动选择添加凭证照片时使用。选取的照片将复制保存到应用私有沙箱目录中，不会上传至任何服务器。
  • 日历读写权限：用于将您设定的资产到期日写入系统日历，由操作系统提供到期提醒。您可以随时在系统设置中关闭此权限。
  • 通知权限：用于在资产到期前按您设定的天数推送本地提醒通知。所有通知均由设备本地生成，不经过任何远程服务器。

以上所有权限均为可选权限，拒绝授权不影响应用基础功能的使用。

三、数据存储与安全

  • 所有数据存储在设备本地关系型数据库中，采用系统级安全保护。
  • 应用不包含任何网络通信模块，无法向外部传输数据。
  • 凭证照片保存在应用沙箱目录中，其他应用无法访问。
  • 卸载应用将彻底删除所有本地数据，不可恢复。

四、第三方 SDK

本应用不集成任何第三方 SDK，包括但不限于：

  • 不包含广告 SDK
  • 不包含数据统计/分析 SDK
  • 不包含推送服务 SDK（通知由系统本地生成）
  • 不包含社交分享 SDK

五、未成年人保护

本应用不针对未成年人提供服务，也不主动收集未成年人的个人信息。由于应用完全本地运行且不收集任何信息，不存在未成年人信息保护风险。

六、隐私政策的变更

我们可能会适时修订本隐私政策。当政策发生变更时，我们会在应用内以弹窗等明显方式通知您。变更后的隐私政策自公布之日起生效。

七、您的权利

  • 您可以随时在系统设置中关闭本应用的各项权限。
  • 您可以随时卸载应用以删除所有本地数据。
  • 您可以通过以下方式联系我们了解隐私相关事宜。
