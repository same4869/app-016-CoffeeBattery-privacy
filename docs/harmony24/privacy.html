读数 隐私政策

最后更新日期：2026年5月7日 | 生效日期：2026年5月7日

读数是一款完全本地化的阅读管理工具。我们不收集、不上传、不存储任何用户数据。

一、我们的承诺

「读数」（以下简称"本应用"）由开发者独立开发和运营。我们深知阅读记录属于个人隐私，因此郑重承诺：

  • 所有数据仅存储在您的设备本地数据库中
  • 不收集任何个人身份信息
  • 不上传任何阅读数据到云端服务器
  • 不集成任何第三方数据分析、广告或追踪 SDK
  • 不追踪您的使用行为或阅读偏好
  • 应用不需要联网即可正常使用所有功能

二、数据存储

本应用使用设备本地数据库存储以下信息：

  • 您录入的书籍信息（书名、作者、分类、来源平台、页数等）
  • 阅读记录（专注时长、阅读页数、阅读配速等）
  • 阅读目标（年度目标、月度书单等）
  • 应用设置（昵称、签名等个性化配置）

以上数据完全存储在您的设备中，开发者无法以任何方式访问。

三、设备权限

本应用可能请求以下设备权限：

  • 后台运行权限：仅用于阅读计时器在应用切入后台时继续计时，确保专注数据不丢失。
  • 相册/媒体读取权限：仅在您主动选择书籍封面图片时触发，用于从相册选取图片作为书籍封面。
  • 文件写入权限：仅在您主动导出阅读数据报告时触发，用于将 JSON 格式的报告保存到您指定的位置。

以上权限均需您明确授权后方可使用，且仅在对应功能操作时触发，不会在后台静默使用。

四、数据导出与删除

  • 您可以随时通过「我的 → 导出读数报告」将所有数据导出为 JSON 文件，便于个人备份。
  • 卸载本应用将永久删除设备上存储的所有数据。
  • 由于我们从未持有您的数据副本，因此无法为您恢复已删除的数据。

五、第三方服务

本应用不集成任何第三方服务，包括但不限于：

  • 不使用第三方统计分析工具
  • 不使用第三方广告平台
  • 不使用第三方推送服务
  • 不使用第三方登录服务

六、未成年人保护

本应用不收集任何用户信息，因此不涉及未成年人个人信息处理。本应用内容适合所有年龄段用户使用。

七、隐私政策更新

我们可能会不时更新本隐私政策。更新后的政策将在本页面发布并标注新的生效日期。建议您定期查阅本政策。继续使用本应用即表示您同意最新版本的隐私政策。
