出门别落下 隐私政策

🎒 纯本地 · 无网络 · 无服务器

📋 概述

「出门别落下」是一款个人出行资产管理工具，由个人开发者独立开发运营。我们从产品设计之初就确立了"数据不出设备"的核心原则，确保您的出行数据完全由您自己掌控。

本应用不收集、不存储、不传输任何用户数据至外部服务器。所有数据仅存在于您的设备本地。

💾 数据存储

  • 存储方式：所有数据通过 HarmonyOS 系统级关系型数据库（RDB）和系统 Preferences 安全存储于设备本地
  • 存储内容：物品信息（名称、图标、类别、健康值、库存数量）、场景信息（名称、图标、关联物品）、出行打卡记录、每日整备评分、用户偏好设置等
  • 存储位置：仅存储在应用沙箱内的本地数据库中，不会上传至任何云端或第三方服务器
  • 数据留存：数据随应用存在而保留，用户可随时通过应用内功能导出或删除数据；卸载应用后数据将被系统自动清除

🚫 数据收集与共享

  • 本应用不收集任何个人身份信息用于商业目的
  • 本应用不使用任何第三方统计 SDK、广告 SDK 或数据分析服务
  • 本应用不与任何第三方共享用户数据
  • 本应用不要求网络权限，不具备联网能力
  • 本应用不使用设备标识符（如 IMEI、OAID、MAC 地址等）

🔐 权限说明

本应用仅申请以下最小必要权限：

  • 文件读写权限（ohos.permission.FILE_ACCESS_PERSIST）：用于数据备份导出与导入恢复功能，由用户主动触发，仅在用户操作时使用

本应用不会申请相机、定位、通讯录、麦克风等敏感权限。

🛡️ 数据安全

  • 采用 HarmonyOS 系统级数据库存储，数据库文件受系统沙箱保护
  • 应用运行在独立沙箱中，其他应用无法访问本应用的数据
  • 不支持云同步功能，数据不会意外同步至云端
  • 数据备份导出功能由用户主动触发，导出文件保存至用户指定位置，由用户自行管理
  • 用户可通过"重置演示数据"功能一键清除全部业务数据

📤 数据导出与删除

  • 导出：用户可随时通过「我的」页的"数据备份与导出"功能将全部数据导出为 JSON 文件
  • 导入：用户可通过"数据导入恢复"功能从备份文件恢复数据
  • 删除：用户可在应用内删除单条记录，或通过"重置演示数据"功能清除全部数据
  • 卸载清除：卸载应用后，系统将自动清除所有本地数据

🔄 政策更新

本隐私政策可能会随应用功能迭代而更新。重大变更将在应用更新时通过弹窗或应用内通知告知用户。继续使用应用即表示您同意更新后的隐私政策。
