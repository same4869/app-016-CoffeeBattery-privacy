隐私政策

六迹 (SixEchoes) | 最后更新：2026年5月

六迹是一款纯本地单机应用。我们深知个人日记的私密性，因此从架构层面确保您的数据安全。以下是我们对您隐私的完整承诺。

一、数据存储

您在六迹中记录的所有内容（文字、心情标记、记录时间）均保存在您设备本地的 SQLite 数据库中。数据不会被上传至任何远程服务器、云端存储或第三方平台。

二、网络访问

六迹不申请网络访问权限。应用在无网络环境下可完整运行。我们不收集、不传输、不共享任何用户数据。

三、权限使用

六迹仅申请以下权限：

  • 本地通知权限 (ohos.permission.PUBLISH_AGENT_REMINDER)：用于"每日提醒"功能。通知内容为固定提示文案（"今天的六个词，想好了吗？"），不包含任何日记内容。此功能为可选功能，不开启不影响使用。

四、数据导出与备份

您可以随时通过"我的 → 导出数据"将全部日记记录导出为 JSON 或 CSV 格式文件，导出文件保存在应用沙箱目录中，您拥有数据的完全所有权。

五、数据删除

  • 应用内可在时光页面删除单条记录
  • 卸载应用将永久清除所有本地数据，此操作不可恢复
  • 建议您在卸载前使用导出功能进行备份

六、第三方服务

六迹不集成任何第三方 SDK、数据分析工具、广告投放服务、用户行为追踪系统或社交分享 SDK。应用内的"分享海报"功能通过系统原生分享接口实现，不经过第三方服务。

七、儿童隐私保护

六迹不面向14岁以下儿童用户，不主动收集任何个人身份信息。由于应用完全本地运行且无网络功能，不存在儿童数据泄露风险。

八、桌面卡片（小组件）

六迹提供桌面卡片功能，卡片数据来源于本地数据库，仅在设备本地渲染显示，不涉及任何数据传输。

九、隐私政策更新

如本政策有实质性变更，我们将在应用内以弹窗形式通知您。继续使用即视为同意更新后的政策。

十、联系我们
