轴迹 隐私政策

🔒 纯本地 · 无网络 · 无服务器

📋 概述

轴迹是一款儿童眼轴发育追踪工具，由个人开发者独立开发运营。我们深知儿童健康数据的敏感性，因此从产品设计之初就确立了"数据不出设备"的核心原则。

本应用不收集、不存储、不传输任何用户数据至外部服务器。所有数据仅存在于您的设备本地。

💾 数据存储

  • 存储方式：所有数据通过 HarmonyOS 系统级关系型加密数据库（Security Level S1）和系统 Preferences 安全存储于设备本地
  • 存储内容：儿童姓名、出生日期、性别、身高、眼轴长度、等效球镜度数、角膜曲率、干预策略标注等测量记录
  • 存储位置：仅存储在应用沙箱内的本地数据库中，不会上传至任何云端或第三方服务器
  • 数据留存：数据随应用存在而保留，用户可随时通过应用内功能导出或删除数据；卸载应用后数据将被系统自动清除

🚫 数据收集与共享

  • 本应用不收集任何个人身份信息用于商业目的
  • 本应用不使用任何第三方统计 SDK、广告 SDK 或数据分析服务
  • 本应用不与任何第三方共享用户数据
  • 本应用不要求网络权限，不具备联网能力
  • 本应用不使用设备标识符（如 IMEI、OAID、MAC 地址等）

🔐 数据安全

  • 采用 HarmonyOS 系统级加密存储（RDB Security Level S1），数据库文件受系统沙箱保护
  • 应用运行在独立沙箱中，其他应用无法访问本应用的数据
  • 不支持云同步功能，数据不会意外同步至云端
  • CSV 导出功能由用户主动触发，导出文件保存至用户指定位置，由用户自行管理

👶 未成年人保护

本应用专为 3-18 岁儿童及青少年设计，数据由家长或监护人录入和管理。我们承诺：

  • 不收集未成年人的任何可识别身份信息用于商业目的
  • 不向未成年人展示任何商业广告或营销内容
  • 不提供任何社交功能，不存在信息泄露至他人的风险
  • 数据解读结果仅供参考，不构成医疗诊断或治疗建议

📤 数据导出与删除

  • 导出：用户可随时通过「档案」页的导出功能将全部数据导出为 CSV 文件
  • 删除：用户可在应用内删除单条记录或重置全部数据
  • 卸载清除：卸载应用后，系统将自动清除所有本地数据

⚠️ 免责声明

本应用仅提供数据记录与趋势分析功能，所有评估结果（包括眼轴年化增速评估、综合发育评估、AI 诊断建议等）均基于数据趋势的自动化计算，仅供参考，不构成任何医疗诊断或治疗建议。儿童的近视防控方案应由专业眼科医生根据全面检查结果制定。

🔄 政策更新

本隐私政策可能会随应用功能迭代而更新。重大变更将在应用更新时通过弹窗或应用内通知告知用户。继续使用应用即表示您同意更新后的隐私政策。
